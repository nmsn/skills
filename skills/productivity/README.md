# Productivity

Workflow tools that are not strictly engineering — GitHub consumption, issue triage, and information aggregation.

- **[follow-frontend](./follow-frontend/SKILL.md)** — Aggregate frontend tech news from RSS sources (dev.to, Hacker News, JS Weekly, Node Weekly, React, CSS-Tricks).
- **[fork-issue-workflow](./fork-issue-workflow/SKILL.md)** — Scan local fork directories for easy upstream GitHub issues and drive them through fix → review → PR.
- **[github-daily-digest](./github-daily-digest/SKILL.md)** — Daily summary of followed GitHub users' commit activity.
- **[github-project-history](./github-project-history/SKILL.md)** — Analyze a GitHub repository's development history: commits, tags, releases, milestones, contributors.
- **[issue-analyzer](./issue-analyzer/SKILL.md)** — Fetch GitHub issues, assess difficulty, and locate the code that needs to change.
